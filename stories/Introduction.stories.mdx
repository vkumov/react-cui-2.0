# React Cisco UI Kit library

You need to download Cisco UI kit and put CSS and font files into folder of your project (`/public/` for example).
Import `cui-standard.min.css` in your code directly (for NextJS).

Bootstrap code:
```jsx
import React from "react";
import { Portal, ToastContainer, ModalProvider } from "@vkumov/react-cui-2.0";

import "../public/css/cui-standard.min.css";
// importing additional styles
// is is required to import both cui-standard.min.css and styles from this package
import "@vkumov/react-cui-2.0/css/styles.css";

const App = () => {
  return <>
    <ModalProvider>
      {/* Your code */}
    </ModalProvider>
    <Portal id="toast-portal">
      <ToastContainer
        pauseOnFocusLoss={false}
        position="top-center"
        autoClose={2000}
      />
    </Portal>
  </>
} 
```
